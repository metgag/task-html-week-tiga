Short Basic HTML

Description, HTML merupakan kependekan dari Hyper Text Markup Language.

  • HTML bukan bahasa pemrogramman, melainkan bahasa markup yang ditampilkan oleh web browser
  • HTML di dukung oleh CSS untuk styling dan JavaScript untuk script.
